Dasar Pemrograman: HTML, CSS, Python, Java, PHP

Halaman ini menjelaskan dasar-dasar pemrograman dan web development yang umum diajarkan di kelas 10 SMK. Masing-masing bagian berisi fungsi, contoh kode, dan penjelasan hasilnya agar lebih mudah dipahami.

📘 HTML (HyperText Markup Language)

HTML digunakan untuk membuat struktur halaman web. Elemen HTML terdiri dari tag seperti <h1> untuk judul, <p> untuk paragraf, dan lainnya. 

<h1>Halo Dunia</h1>
<p>Ini paragraf pertama saya.</p>

🔍 Penjelasan: Kode di atas akan menampilkan judul besar bertuliskan "Halo Dunia" dan satu paragraf di bawahnya. Ini adalah dasar tampilan konten pada web.

🎨 CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan HTML. Kita bisa mengubah warna, ukuran teks, jarak antar elemen, dan banyak lagi. 

h1 {
  color: blue;
  font-size: 24px;
}

🔍 Penjelasan: Kode ini membuat semua elemen <h1> berwarna biru dan berukuran 24 piksel. CSS membantu membuat website menjadi menarik dan profesional.

🐍 Python

Python adalah bahasa pemrograman yang mudah dibaca dan cocok untuk pemula. Sering digunakan untuk analisis data, pengembangan web, dan automasi. 

nama = input("Siapa namamu? ")
print("Halo, " + nama)

🔍 Penjelasan: Program ini meminta pengguna mengetikkan nama, lalu menyapa mereka. Misalnya jika Anda mengetik "Andi", maka akan muncul "Halo, Andi".

☕ Java

Java adalah bahasa OOP (Object-Oriented Programming) yang banyak digunakan di Android dan aplikasi desktop. 

public class Halo {
  public static void main(String[] args) {
    System.out.println("Halo Dunia!");
  }
}

🔍 Penjelasan: Ini adalah program Java paling sederhana. Fungsi main() akan dijalankan pertama kali dan menampilkan teks "Halo Dunia!" ke layar.

🐘 PHP

PHP adalah bahasa pemrograman yang dijalankan di server. PHP sangat populer untuk membuat website dinamis seperti blog, toko online, dan sistem login. 

<?php
$nama = "Andi";
echo "Halo, $nama!";
?>

🔍 Penjelasan: PHP menyimpan nama "Andi" ke dalam variabel $nama, lalu mencetak "Halo, Andi!" ke halaman web.

Dengan memahami kelima dasar ini, siswa memiliki fondasi kuat untuk belajar pemrograman lebih lanjut seperti JavaScript, framework web, dan pengembangan aplikasi nyata.
