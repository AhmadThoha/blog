Materi: Basis Data Lanjutan

Materi ini membahas konsep lanjutan dalam basis data seperti perintah SQL (DDL, DML, DCL), subquery, dan penggunaan JOIN untuk menggabungkan data dari beberapa tabel. Sangat penting untuk kamu yang ingin menjadi programmer atau pengembang aplikasi.

1. DBMS (Database Management System)

DBMS adalah sistem perangkat lunak untuk membuat dan mengelola database. Contoh: MySQL, Oracle, PostgreSQL, SQL Server. DBMS memungkinkan pengguna untuk menyimpan, mengakses, mengubah, dan menghapus data secara efisien.

2. DDL (Data Definition Language)

  • CREATE – Membuat tabel atau objek database.
  • ALTER – Mengubah struktur tabel (menambah/ubah kolom).
  • DROP – Menghapus tabel atau objek.

3. DML (Data Manipulation Language)

  • SELECT – Mengambil data dari tabel.
  • INSERT – Menambahkan data ke tabel.
  • UPDATE – Mengubah data di tabel.
  • DELETE – Menghapus data dari tabel.

4. DCL (Data Control Language)

  • GRANT – Memberi hak akses kepada pengguna.
  • REVOKE – Menghapus hak akses dari pengguna.

5. Subquery (Query Bersarang)

Subquery adalah query di dalam query lain. Contoh: 

SELECT * FROM produk 
WHERE harga > (SELECT AVG(harga) FROM produk);

Penjelasan: Menampilkan produk yang harganya di atas rata-rata.

6. JOIN (Menggabungkan Tabel)

JOIN digunakan untuk menggabungkan data dari dua atau lebih tabel yang memiliki relasi.

  • INNER JOIN: Menampilkan data yang cocok di kedua tabel.
  • LEFT JOIN: Menampilkan semua data dari tabel kiri dan yang cocok dari tabel kanan.
  • RIGHT JOIN: Menampilkan semua data dari tabel kanan dan yang cocok dari tabel kiri.
SELECT pelanggan.nama, transaksi.total
FROM pelanggan
INNER JOIN transaksi ON pelanggan.id = transaksi.pelanggan_id;

Penjelasan: Menampilkan nama pelanggan beserta total transaksi mereka.
